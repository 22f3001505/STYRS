# ⚡ STYRS — Solar Cell Defect Detection

AI-powered quality inspector for solar cell manufacturing.

## Features
- 🧠 **89.24% Accuracy** with EfficientNetB3
- 📊 Single & batch image analysis
- 📄 PDF report generation
- 📈 Confidence visualization with gauges

## Model
- **Architecture**: EfficientNetB3
- **Input**: 300×300 RGB images
- **Classes**: Defective / Good
- **Training**: 6 combined datasets with progressive unfreezing
